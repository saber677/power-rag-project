# 支付系统

## 支持的支付方式

- 支付宝（APP支付、H5支付、扫码支付）
- 微信支付（JSAPI、Native、APP）
- 银联云闪付
- 余额支付（平台钱包）

## 支付流程

1. 用户提交订单，选择支付方式
2. 后端创建支付单，调用第三方支付接口获取支付凭证
3. 前端拉起支付（或展示二维码）
4. 用户完成支付
5. 第三方回调通知 → 更新支付单状态 → 更新订单状态
6. 如未收到回调，定时任务每5分钟主动查询支付结果

## 退款规则

- 全额退款：订单未发货时支持全额退款
- 部分退款：已发货订单，退部分商品金额
- 退款到账时间：支付宝/微信 1-3个工作日，银联 3-5个工作日
- 余额支付退款实时到账

## 对账机制

每日凌晨2点执行对账：
1. 拉取第三方支付账单
2. 与本地支付记录逐笔比对
3. 差异记录写入异常表，人工处理
4. 生成对账报告发送至财务邮箱
