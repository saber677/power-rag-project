# 用户中心

## 用户注册

支持以下注册方式：
- 手机号 + 验证码
- 邮箱 + 密码
- 第三方登录（微信、支付宝、钉钉）

注册流程：
1. 用户输入手机号/邮箱
2. 发送验证码（60秒冷却）
3. 验证通过后创建账户
4. 自动分配默认角色（普通用户）

## 用户等级体系

| 等级 | 名称 | 累计消费 | 权益 |
|------|------|----------|------|
| L1 | 普通会员 | 0 | 基础服务 |
| L2 | 银卡会员 | ≥1000元 | 95折优惠 |
| L3 | 金卡会员 | ≥5000元 | 9折优惠 + 专属客服 |
| L4 | 钻石会员 | ≥20000元 | 85折 + 专属客服 + 优先发货 |

等级每月1号重新计算，基于近12个月累计消费金额。

## 账户安全

密码规则：
- 最少8位，包含大小写字母和数字
- 不允许与最近5次密码相同
- 连续输错5次锁定账户30分钟

Token 机制：
- 登录后签发 JWT Token，有效期2小时
- Refresh Token 有效期7天
- 单设备登录，新登录会踢掉旧设备
